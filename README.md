# AI-Based Crop Yield Prediction

## Overview

This project predicts crop yield using machine learning techniques based on environmental and agricultural factors such as temperature, rainfall, and humidity.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn

## Features

* Predict crop yield based on input data
* Data preprocessing and analysis
* Basic visualization support

## Input Parameters

* Temperature
* Rainfall
* Humidity

## Output

* Predicted crop yield value

## Project Structure

* app.py: Main application file
* model.py: Machine learning logic
* data/: Dataset files

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the application:
   python app.py
